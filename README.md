# Atividade 0 - Algoritmos

Este repositório contém as soluções da **Atividade 0** da disciplina de **Algoritmos**, organizadas por tema: **arrays**, **structs** e **unions**.

## Organização dos Arquivos

- **arrays_exercises/**
  - `exercise_1.cpp`: Manipulação de temperaturas da semana usando arrays.
  - `exercise_2.cpp`: Cálculo da média de números inteiros fornecidos pelo usuário.
  - `exercise_3.cpp`: Inversão de uma palavra fornecida pelo usuário.

- **structs_exercises/**
  - `exercise_1.cpp`: Struct `Livro` com título, autor e ano de publicação.
  - `exercise_2.cpp`: Cálculo da distância euclidiana entre dois pontos usando struct `Ponto`.
  - `exercise_3.cpp`: Array de structs `Aluno` para manipular dados de alunos.

- **unions_exercises/**
  - `exercise_1.cpp`: Union `Valor` para demonstrar sobrescrita de dados.
  - `exercise_2.cpp`: Union `EnderecoIP` para manipular endereços IP em diferentes formatos.
  - `exercise_3.cpp`: Struct contendo uma union e um enum para rastrear tipos de dados.
